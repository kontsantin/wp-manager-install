# WordPress Manager

Интерактивный менеджер для создания и управления проектами WordPress с использованием Docker.

## Возможности

- **Создание проектов**: Автоматическое создание WordPress-сайта с Docker, базовой темой и полными исходниками.
- **Управление проектами**: Запуск, остановка, удаление проектов.
- **Открытие в VS Code**: Выбор открытия всего сайта или только темы/плагина.
- **Очистка**: Полное удаление проектов, включая БД и домены из hosts.

## Установка

1. Скачайте скрипт:
   ```bash
   wget https://raw.githubusercontent.com/kontsantin/wp-manager-install/main/wpmanager
   chmod +x wpmanager
   sudo mv wpmanager /usr/local/bin/
   ```

2. Установите зависимости:
   - Docker Desktop
   - VS Code (с расширением WSL, если в WSL)

## Использование

Запустите:
```bash
wpmanager
```

Выберите действие из меню:
1. Создать новый проект
2. Список проектов
3. Запустить проект
4. Остановить проект
5. Удалить проект
6. Открыть проект в VS Code
7. Открыть сайт в браузере

## Создание проекта

- Введите название проекта (станет темой).
- Укажите домен (по умолчанию `название.local`).
- Скрипт создаст:
  - Docker-контейнеры (WordPress + MariaDB)
  - Полные исходники WordPress
  - Базовую тему в `wp-content/themes/название/`
  - Записи в hosts

## Открытие в VS Code

При выборе "6" выберите:
- Весь сайт (исходники + wp-content)
- Только wp-content
- Только тема проекта

## Удаление проекта

Полностью удаляет контейнеры, volumes, файлы и домены.

## Требования

- Linux/WSL с Docker
- Bash
- sudo для некоторых операций
